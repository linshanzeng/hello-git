# Git教程

## 远程仓库

### 从远程仓库克隆

克隆一个本地库：`git@github.com:linshanzeng/hello-git2.git`

## 分支管理

创建dev分支：

```text
方法一：
创建dev分支，并切换到dev分支：git checkout -b dev
方法二：
创建dev分支：git branch dev
切换到dev分支：git checkout dev
```

查看当前分支：`git branch`
